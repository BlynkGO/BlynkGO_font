## ประสานมิตร prasanmit
BlynkGO ได้รวม ฟอนต์ประสานมิตร ขนาด 20, 25, 30, 35, 40 ไว้อยู่แล้ว   
ผู้ใช้ สามารถเรียกใช้ได้เลยโดยไม่ต้องประกาศ `FONT_DECLARE(...);` อีก
- prasanmit_20 
- prasanmit_25 
- prasanmit_30 
- prasanmit_35 
- prasanmit_40

หากต้องการเรียกใช้ ให้ประกาศ `FONT_DECLARE(...);` ก่อนใช้
